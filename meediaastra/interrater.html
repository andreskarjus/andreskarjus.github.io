Inter-Rater Agreement Report

Multi-label matrices ignore label order. For set-valued variables, diagonal cells show shared labels, off-diagonal cells show non-overlapping labels, and [NONE] means the empty set / explicit NA. Soft overlap is pairwise Jaccard over unordered sets.

Jaccard interpretation: 1.0 means the two raters chose the same unordered set, 0.0 means they shared no labels, and values in between indicate partial overlap. Cohen's kappa is reported as a stricter chance-corrected measure.
